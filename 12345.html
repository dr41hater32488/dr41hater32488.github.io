Public Beta

Below are the public profiles, these profiles are usually released later than Developer Beta, so it will be stable and less buggy. If you've never signed up for Apple's Public Beta Software Program before, sign up first at beta.apple.com.
Learn how to switch from Developer beta to Public beta updates

Important Note

If you are installing a beta profile that automatically deletes itself after installation, you are not alone, as Apple is taking actions to completely drop beta profiles.

Currently, you can still use IPSW files to update to the Developer beta, which can be easily found with the keyword "ipsw beta". Just download the .ipsw file for your device and update using iTunes.

How to use a

Beta Profile

1. Archive a backup

1. Archive a backup

Always back up your data and files before installing beta versions. For iPhone and iPad, we recommend that you do a backup using Finder.

Learn how to backup your iPhone and iPad

Learn how to backup your Mac

2. Install profile

2. Install profile

First, check if your device supports updating to the version you want at the Compatibility List page.

On supported devices, go to BetaProfiles.dev to download the configuration profile. Open Settings, tap the profile and follow the onscreen instructions to allow installation.

3. Update your device

3. Update your device

Tap Settings > General > Software Update to install any available beta software.

AirPods Beta firmware details

AirPods beta firmware for Apple Developer Program members enables development of features on iOS and macOS for AirPods. This program also enables debugging of issues by Apple with on-in log collection. This release includes improvements to Automatic Switching and various bug and stability fixes.

Apple doesn't release any beta profiles for AirPods at this time, you need to activate it through Xcode instead.
